
# High Risk Pregnancy Detection

## Overview
This project focuses on identifying high-risk pregnancies using machine learning techniques, specifically Support Vector Machines (SVM), based on the methodologies presented in the referenced research paper.

## Problem Statement
Identification of High Risk Pregnancy

## References
- **Base Paper**: [Sciencedirect Article](https://www.sciencedirect.com/science/article/pii/S0952197625028830)
- **Dataset**: [Consorcio Madrono Dataset](https://edatos.consorciomadrono.es/dataset.xhtml?persistentId=doi:10.21950/WDZX91)

## Project Structure
- **Implementation1**: SVM implementation according to the base paper
- **outputs**: Results and outputs from the SVM model

## Requirements
```
numpy
pandas
scikit-learn
matplotlib
seaborn
scipy
imbalanced-learn
gdown

```

## Usage
Run the SVM implementation:
```bash
python3 Implementation1/Implementation_1.py
```
